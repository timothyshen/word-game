# 诸天文字游戏 - 设计文档

## 概述

回合制文字策略游戏，结合卡牌、领地经营、多世界探索。玩家每日有固定行动点，通过打出卡牌执行操作，经营领地、探索诸天世界、收集角色、解锁隐藏剧情。

## 技术栈

- 前端：Next.js + React + Tailwind
- 后端：tRPC + Prisma + NextAuth
- 数据库：SQLite（开发）/ PostgreSQL（生产）
- 内容：Markdown（剧情）+ JSON（数值数据）

## 核心系统

### 1. 核心循环

```
玩家随时游玩 → 消耗体力执行行动
    ↓
城内建设（消耗卡牌）/ 城外冒险（消耗体力）
    ↓
体力自然回复 / 获得卡牌掉落
    ↓
每日0点结算 → 计算分数 → 奖励卡牌
```

### 2. 体力系统

**基础设定**：
- 初始上限：100点
- 回复速率：每X分钟回复1点
- 可升级：提升上限 + 提升回复效率

**体力消耗**：

| 行动 | 体力消耗 | 说明 |
|------|---------|------|
| 探索新区域 | 15-30 | 根据区域难度 |
| 采集资源点 | 5-10 | 根据资源稀有度 |
| 战斗小怪 | 10-20 | 根据怪物等级 |
| 挑战Boss | 30-50 | 高消耗高回报 |
| 城内建设 | 0-5 | 主要消耗卡牌 |

**体力来源**：
- 自然时间回复（主要）
- 体力药水（道具）
- 每日登录奖励
- 特定建筑效果

### 3. 结算系统

**结算时间**：每日 0:00 自动结算

**行动分数计算**：
根据当天（0:00-23:59）的行动计算总分数：

| 行动类型 | 基础分数 | 加成条件 |
|---------|---------|---------|
| 建造建筑 | +50 | 首次建造+20 |
| 升级建筑 | +30 × 等级 | 连续升级+10 |
| 完成探索 | +40 | 发现新区域+30 |
| 击败敌人 | +20 × 难度 | 无伤+50% |
| 完成任务 | +100 | 支线+50, 隐藏+200 |
| 招募角色 | +60 | 稀有度加成 |
| 生产产出 | +10 | 超额产出+20% |

**分数奖励**：
```
分数区间        奖励
0-99           1张普通卡
100-199        2张普通卡
200-299        2张普通卡 + 1张精良卡
300-399        3张普通卡 + 1张精良卡
400-499        2张精良卡 + 1张稀有卡
500+           3张精良卡 + 1张稀有卡 + 随机奖励
```

**连续奖励**：
- 连续3天达成200+分：额外获得1张稀有卡
- 连续7天达成300+分：获得1张史诗卡
- 单日达成1000+分：成就「传奇一日」+ 传说卡抽卡券

### 2. 卡牌系统

**卡牌类型**：

| 类型 | 作用 | 示例 |
|------|------|------|
| 建筑卡 | 在领地建造建筑 | 农田、兵营、铁匠铺 |
| 招募卡 | 获得下属角色 | 流浪剑士、村民、神秘商人 |
| 提升卡 | 升级建筑/角色 | 强化石、技能书 |
| 物品卡 | 消耗品或装备 | 回复药、武器、探索道具 |
| 技能卡 | 角色战斗/生产技能 | 火球术、采矿术、治疗术 |

**获取途径**：

| 来源 | 说明 | 稳定性 |
|------|------|-------|
| 每日结算 | 0点结算，根据当日分数奖励 | 稳定 |
| 打野(小怪) | 城外战斗随机掉落 | 随机 |
| Boss | 每周重置，随机出现，稀有卡 | 周期性 |
| 卡牌祭坛 | 城内建筑，抽卡/合成/献祭 | 可控 |
| 野外商人 | 实时刷新，资源购买 | 随机 |
| 剧情奖励 | 主线/支线/隐藏剧情 | 一次性 |

**卡牌祭坛功能**：
- **抽卡**：消耗资源随机抽取卡牌
- **合成**：多张低级卡合成高级卡
- **献祭**：销毁卡牌换取其他奖励（资源/经验/特殊道具）

### 3. 地图系统

地图分为 **城内** 和 **城外** 两个区域：

```
┌─────────────────────────────────────────────┐
│                   城外（冒险区）              │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐        │
│  │野外 │  │矿洞 │  │森林 │  │遗迹 │ ...    │
│  │设施 │  │入口 │  │深处 │  │探索 │        │
│  └─────┘  └─────┘  └─────┘  └─────┘        │
│                    ↓ 城门 ↓                  │
│  ┌─────────────────────────────────────┐    │
│  │              城内（建设区）           │    │
│  │  ┌────┐ ┌────┐ ┌────┐ ┌────┐       │    │
│  │  │农田│ │兵营│ │主城│ │市场│ ...   │    │
│  │  └────┘ └────┘ └────┘ └────┘       │    │
│  └─────────────────────────────────────┘    │
└─────────────────────────────────────────────┘
```

**城内（建设区）**：
- 建造和升级建筑
- 分配角色工作
- 管理资源产出
- 安全区域，无战斗

**城外（冒险区）**：
- 探索未知区域
- 战斗和收集资源
- 发现野外设施
- 每日结算后刷新部分内容

**野外设施（实时刷新）**：
城外设施实时刷新，不依赖每日结算：

| 设施类型 | 说明 | 刷新规则 |
|---------|------|---------|
| 资源点 | 采集木材/矿石/草药 | 采集后消失，30-60分钟后随机刷新 |
| 小怪 | 普通战斗，掉落卡牌 | 击杀后10-30分钟刷新 |
| 野外商人 | 用资源购买卡牌 | 随机出现，停留2-4小时后离开 |
| Boss | 稀有卡牌掉落 | 随机出现，每周重置挑战次数 |
| 神秘遗迹 | 探索解谜 | 完成后永久消失 |
| 传送裂隙 | 通往其他世界 | 随机出现，开放1-2小时 |

**Boss机制**：
- 随机在城外区域出现
- 可单人挑战或多角色编队
- 每周一0点重置挑战次数
- 掉落稀有/史诗卡牌

### 4. 领地系统

```
城内建筑分类
├── 核心区（主城堡，决定领地等级上限）
├── 生产区（农田、矿场、伐木场）
├── 军事区（兵营、训练场、城墙）
├── 商业区（市场、仓库、银行）
└── 特殊区（传送门、研究所 - 解锁后出现）
```

- 每日结算时根据建筑等级 + 分配的角色 + 角色技能计算产出
- 部分建筑需要消耗资源运转
- 特殊建筑解锁新玩法（如传送门解锁世界穿梭）

### 4. 诸天世界

```
主位面（玩家领地所在）
│
├── 仙侠世界 - 灵石、仙草、修炼功法
├── 末日世界 - 废土资源、变异材料、科技残片
├── 中世纪世界 - 铁矿、骑士、魔法卷轴
├── 东瀛世界 - 神社、妖怪、武士道
└── ... 更多世界后续解锁
```

**穿梭机制**：
- 需要建造「传送门」建筑
- 穿梭消耗行动点 + 特定资源
- 每个世界有独特资源、敌人、剧情线
- 部分稀有职业只能在特定世界解锁

### 5. 探索与战斗

**探索流程**：
```
选择世界 → 选择区域 → 消耗行动点进入
    ↓
遭遇事件（文字描述 + 选项）
    ↓
├── 战斗 → 文字选择战斗 → 胜利/失败结算
├── 对话 → 影响好感/触发支线
├── 发现 → 获得资源/卡牌/线索
└── 陷阱 → 损失/考验
```

**文字选择战斗示例**：
```
【遭遇：山贼头目】
"一个身披破旧铠甲的壮汉挡住去路，手持大刀..."

HP: 你 80/100 | 敌 120/120

> 1. 正面强攻（消耗：剑气斩卡）
> 2. 尝试说服（需要：口才 > 15）
> 3. 投掷烟雾弹撤退（消耗：烟雾弹卡）
> 4. 召唤下属助战（需要：空闲角色）
```

### 6. 角色成长系统

**双轨制成长**：职阶 + 职业 独立发展

```
┌─────────────────────────────────────────────────────────┐
│  【职阶】基础能力上限          【职业】技能方向特化      │
├─────────────────────────────────────────────────────────┤
│  1阶(6槽) → 2阶(12槽) → 3阶(18槽) → ...               │
│      ↓          ↓           ↓                          │
│   可开启     可开启       可开启                        │
│   新职业     新职业       新职业                        │
│                                                         │
│  无职业 → 剑客 → 剑圣      ← 职业突破卡(动态生成)      │
│            ↓      ↘                                     │
│          刺客    剑魔                                   │
│                    ↘                                    │
│                   职业融合卡 → 融合职业(如:魔剑士)      │
└─────────────────────────────────────────────────────────┘
```

**初始状态**：
- 新角色：1阶 + 无职业
- "领主"只是头衔，非职业

**职阶系统**：

| 职阶 | 技能槽 | 突破条件 |
|-----|-------|---------|
| 1阶 | 6槽 | 初始 |
| 2阶 | 12槽 | 完成1阶突破任务 |
| 3阶 | 18槽 | 完成2阶突破任务 |
| ... | +6槽/阶 | 对应突破任务 |

- 每阶可解锁新的职业选项

**职业系统**：
- **职业突破卡**：基于已有技能**动态生成**可选职业
- **职业融合卡**：融合多个职业特性，创造融合职业
- 职业选项随职阶提升而扩展

**角色属性**：
```
角色属性：
├── 职阶（1阶/2阶/3阶...）
├── 职业（无/剑客/术士/魔剑士...）
├── 基础属性（力量、敏捷、智力、魅力）
├── 技能槽（职阶×6，学习后永久）
├── 装备槽（11个槽位）
├── 好感度（影响忠诚、触发专属剧情）
└── 状态（空闲/工作中/探索中/受伤）
```

**角色操作**：
- **切换视角** → 用不同角色探索城外
- **分配工作** → 安排到建筑提升产出
- **组队探索** → 多角色配合挑战Boss
- **闲置休息** → 恢复状态/触发随机事件

| 部位 | 槽位名 | 主要属性 |
|------|--------|---------|
| 主手 | mainHand | 攻击力、技能加成 |
| 副手 | offHand | 防御/攻击、特殊效果 |
| 头盔 | helmet | 防御、特殊抗性 |
| 胸甲 | chest | 防御、生命值 |
| 腰带 | belt | 负重、物品槽 |
| 护手 | gloves | 攻击速度、暴击 |
| 裤子 | pants | 防御、移动速度 |
| 鞋子 | boots | 移动速度、闪避 |
| 项链 | necklace | 特殊属性、法力 |
| 戒指1 | ring1 | 属性加成 |
| 戒指2 | ring2 | 属性加成 |

**装备品质**：
```
普通(白) → 精良(绿) → 稀有(蓝) → 史诗(紫) → 传说(橙)
```

**装备来源**：
- 铁匠铺锻造
- 探索掉落
- 商店购买
- 任务奖励
- 卡牌获取

### 8. 技能与战斗系统

**技能系统**：
```
技能卡 ──→ 学习 ──→ 永久掌握技能（消耗卡牌）
                        ↓
               占用技能槽（上限=职阶×6）
                        ↓
               战斗中可使用（有冷却）
```

**技能升级**：
- 方式A：使用相同技能卡再次学习 → 技能Lv+1
- 方式B：使用通用「技能升级卡」→ 技能Lv+1

**战斗系统**（文字选择制）：
```
【遭遇：山贼头目】
"一个身披破旧铠甲的壮汉挡住去路..."

HP: 你 80/100 | 敌 120/120

> 1. 剑气斩 [可用] ← 已学技能
> 2. 铁壁 (冷却: 2回合)
> 3. 普通攻击 [始终可用]
> 4. 使用道具 → 消耗道具卡
> 5. 撤退
```

**战斗卡牌分类**：

| 类型 | 使用方式 | 消耗时机 |
|-----|---------|---------|
| 技能卡 | 学习后永久 | 学习时消耗，使用有冷却 |
| 道具卡 | 战斗中即时 | 使用时消耗（一次性） |

**道具卡类型**：
- 回复类：血药、蓝药、解毒剂
- 战术类：烟雾弹、闪光弹、陷阱
- 伤害类：炸弹、毒瓶、符咒
- 增益类：力量药水、护盾卷轴

### 9. 隐藏内容

**触发方式**：
- 条件触发（拥有某卡牌、到达某地点、特定属性值）
- 选择分支（在普通剧情中做出特定选择序列）
- 探索发现（在特定世界/区域随机或固定位置发现线索）
- 组合以上

**示例**：
```
【稀有职业：时空行者】
解锁条件（玩家不可见）：
- 在三个不同世界各完成一条支线
- 收集「破碎时钟」碎片 x5
- 在主位面传送门处选择「凝视裂缝」
```

## 数据模型

### 核心实体

```
User (玩家账号)
├── Player (主角色 - 可操作)
│   ├── 职业（初始无，通过突破卡获得）
│   ├── 等级、属性
│   ├── 装备（11槽位）
│   ├── 行动点（每日重置）
│   └── 当前所在世界/位置
├── Territory (领地)
│   ├── 建筑槽位
│   ├── 资源产出
│   ├── 经济数据
│   └── 军事数据
├── CardInventory (卡牌背包)
├── CharacterRoster (角色列表 - 均可操作)
│   ├── 基础属性
│   ├── 职业（可通过突破卡获得）
│   ├── 装备（11槽位）
│   └── 技能列表
├── Equipment (装备库存)
└── StoryProgress (剧情进度/解锁状态)
```

### Prisma 模型（核心）

```
Player - 玩家存档、进度、状态
Card - 卡牌模板（从 JSON 同步）
PlayerCard - 玩家拥有的卡牌实例
Character - 角色模板
PlayerCharacter - 玩家拥有的角色实例（可操作）
Equipment - 装备模板
PlayerEquipment - 玩家拥有的装备实例
CharacterEquipment - 角色装备关联（11槽位）
Profession - 职业模板
CharacterProfession - 角色职业关联
Building - 建筑模板
PlayerBuilding - 玩家已建造建筑
EconomyLog - 经济数据记录
MilitaryStatus - 军事状态
StoryProgress - 剧情节点完成状态
UnlockFlag - 解锁标记（用于隐藏内容判断）
```
