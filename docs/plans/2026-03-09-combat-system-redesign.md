# 战斗系统重设计

## 概述

将当前简陋的1v1回合制战斗系统升级为两层战斗体系：
1. **3人小队ATB战斗** — 核心战斗系统，用于探索、精英、Boss
2. **军团回合指挥战斗** — 中后期解锁，用于领地攻防和大规模战争

## 一、3人小队ATB战斗

### 1.1 核心循环

- 3名角色组成小队参战
- 每个角色和敌人各有一个 **ATB速度条**，从0开始按速度属性填充
- ATB满的单位获得行动权，行动后清零重新填充
- 速度快的角色可能在一轮内行动两次
- **半即时制**：ATB自动填充，满后暂停等待玩家输入（可选全即时模式）

### 1.2 行动选择

| 行动 | 说明 | 消耗 |
|------|------|------|
| 攻击 | 普通物理攻击 | 0 MP |
| 技能 | 从已学技能列表选择 | 各技能不同 |
| 防御 | 减伤50%，ATB充能加速20% | 0 MP，回复10 MP |
| 道具 | 使用消耗品 | 消耗道具 |
| 逃跑 | 仅队长可发动，Boss战不可用 | - |

### 1.3 目标选择

- 单体技能：选择一个敌人或队友
- 群体技能：自动作用于全体敌人或全体队友
- 攻击默认选择敌方，治疗默认选择己方

### 1.4 战斗结束

- **胜利**：所有敌人倒下
- **失败**：3名角色全部倒下（"力竭"状态，战后自动恢复1HP）

## 二、技能系统

### 2.1 职业技能树（基础层）

每个角色有主职业，职业决定基础技能树。技能树分3条路线，每条4-5个技能。升级消耗技能点（升级获得）。

示例 — 战士职业：
- **破甲路线**：重击 → 穿刺 → 破甲斩 → 致命一击
- **守护路线**：铁壁 → 嘲讽 → 反击 → 不动如山
- **战意路线**：战吼 → 蓄力 → 狂暴 → 背水一战

鼓励专精1-2条路线，形成差异化build。

### 2.2 装备/卡牌补充技能（扩展层）

角色有 **2-3个额外技能槽**（随职阶突破增加），可装备：
- **技能卡牌** — 抽卡、战斗掉落获得
- **装备附带技能** — 某些装备自带被动或主动技能

角色技能构成 = 职业技能树选择 + 装备技能槽

### 2.3 MP管理

- 战斗开始时MP满值
- 不会自然回复
- 通过防御（+10MP）、道具、特定技能回复
- MP是需要管理的战术资源

## 三、元素与属性克制

### 3.1 六元素体系

物理、火、冰、雷、光、暗

### 3.2 克制规则

- 每种怪物有弱点和抗性
- 打中弱点伤害 ×1.5
- 打中抗性伤害 ×0.5
- 鼓励根据敌人调整队伍和技能搭配

## 四、敌人与遭遇设计

### 4.1 普通遭遇

- 探索时随机触发
- 2-4只普通怪物组合
- 类型根据当前世界/区域决定
- 可能出现配合组合（治疗型 + 攻击型）
- 难度根据队伍等级动态缩放

### 4.2 精英遭遇

- 区域探索低概率触发
- 1-2只精英怪，属性显著高于普通怪
- 每个精英有1-2个特殊机制：
  - 反射护盾：每3回合触发，反弹物理伤害
  - 召唤：血量低于50%时召唤小怪
  - 狂暴：每损失25%血量攻击力提升
- 掉落更好的装备和稀有技能卡牌

### 4.3 Boss战

- 多阶段战斗，每阶段Boss行为模式改变
- 阶段转换时有特殊演出文字
- 可能有特殊胜利条件（如"10回合内击破护盾"）
- Boss战不可逃跑

## 五、战斗奖励与评价

### 5.1 奖励

| 类型 | 普通战 | 精英战 | Boss战 |
|------|--------|--------|--------|
| 经验值 | 基础 | ×2 | ×5 |
| 金币 | 基础+随机 | ×2 | ×5 |
| 材料 | 基础材料 | 稀有材料 | 珍贵材料 |
| 技能卡 | 极低概率 | 中概率 | 高概率 |
| 装备 | 低概率 | 中概率 | 必掉 |

参战角色均分经验，未倒下角色额外+20%。

### 5.2 战斗评价

每场战斗评价 S/A/B/C：
- 回合数（越少越好）
- 队伍存活人数
- 利用属性克制次数
- 是否触发连携技

评价倍率：S=1.5x, A=1.2x, B=1.0x, C=0.8x

### 5.3 连携技（Combo）

两个特定角色在同一轮先后行动并使用特定技能时触发：
- 战士"破甲斩" + 法师"火焰术" = "烈焰穿刺"
- 需要玩家发现和解锁

## 六、军团与兵种系统

### 6.1 解锁条件

达到一定等级或完成特定主线剧情后解锁，前期以小队战斗为核心。

### 6.2 五大兵种及进阶

| 兵种类 | 基础 | 进阶1 | 进阶2 | 进阶3 |
|--------|------|--------|--------|--------|
| 步兵 | 民兵 | 重装步兵 | 剑圣 | 皇家近卫 |
| 弓兵 | 猎手 | 长弓手 | 弩兵 | 神射手 |
| 骑兵 | 轻骑 | 重骑士 | 龙骑兵 | 圣骑士 |
| 法师 | 学徒 | 元素师 | 祭司 | 大贤者 |
| 攻城 | 投石手 | 攻城锤 | 火炮兵 | 魔导炮 |

### 6.3 克制关系

- 步兵 → 弓兵 → 骑兵 → 步兵（基础三角）
- 法师克重甲（步兵/重骑）
- 攻城克防御工事，弱于骑兵突袭

### 6.4 军团战斗流程

1. **战前**：编排部队阵型，分配英雄统领（每个英雄带一支部队）
2. **每回合**：给各部队下指令 — 进攻/防御/冲锋/包抄/撤退
3. **英雄行动**：统领英雄释放技能，影响范围内部队
4. **结算**：兵种克制 × 数量差 × 英雄加成 × 地形综合计算伤亡

### 6.5 兵种获取

- **建筑决定类型**：兵营→步兵，射箭场→弓兵，马厩→骑兵，法师塔→法师，工坊→攻城
- **招募消耗资源**：金币 + 对应材料
- **升级建筑**解锁进阶兵种
- 存活部队有经验值，会成长变强

## 八、与现有系统关联

| 现有系统 | 关联方式 |
|----------|----------|
| 角色招募 | 拥有多角色，针对敌人换人上阵 |
| 装备系统 | 11槽位不变，影响战斗属性更直接 |
| 职阶突破 | 解锁更多技能树节点和额外技能槽 |
| 传送门 | 不同世界敌人属性不同，需不同策略 |
| 建筑系统 | 新增军事建筑，产出和升级兵种 |
| 锻造系统 | 战斗材料用于锻造装备 |
| 卡牌系统 | 技能卡牌可装备到额外技能槽 |

### Phase 3：军团系统

11. 军团战斗服务
